### Run Locally

Clone the project

```bash
  git clone https://github.com/himanegi/hitch-hike-2.0.git
```

Go to the project directory

```bash
  cd project-directory
```

Install dependencies

```bash
  npm i
```

Start the client

```bash
  npm run dev
```
